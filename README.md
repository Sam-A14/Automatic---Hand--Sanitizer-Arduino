# Smart IoT-Based Automatic Hand Sanitizer System

## 📌 Overview

This project is an IoT-based automatic hand sanitizer system designed to provide a touch-free and hygienic solution for sanitization. It uses an ultrasonic sensor to detect hand presence and activates a pump mechanism to dispense sanitizer automatically.

> This project is adapted from an existing repository and further enhanced with improvements and future scalability in mind.

---

## ⚙️ Features

* Touch-free sanitizer dispensing
* Ultrasonic sensor-based hand detection
* Automatic pump activation using relay module
* Hygienic and low-cost solution
* Scalable for IoT-based monitoring systems

---

## 🛠️ Components Used

* Arduino / NodeMCU
* Ultrasonic Sensor (HC-SR04)
* Relay Module
* Water Pump / Motor
* Power Supply

---

## 🔄 Working Principle

1. The ultrasonic sensor continuously measures distance.
2. When a hand is detected within a threshold range,
3. The microcontroller triggers the relay module.
4. The relay activates the pump.
5. Sanitizer is dispensed automatically.

---

## 🚀 How to Run

1. Clone or download the repository
2. Open the code in Arduino IDE
3. Select the correct board (Arduino/NodeMCU)
4. Connect hardware as per circuit
5. Upload the code
