# Automatic Hand Sanitizer

## Description
This is an Arduino-based project that automatically dispenses sanitizer using an ultrasonic sensor.

## Components Used
- Arduino / NodeMCU
- Ultrasonic Sensor
- Relay Module
- Pump / Motor

## Working
When a hand is detected by the ultrasonic sensor, the relay activates the pump and sanitizer is dispensed automatically.

## How to Run
1. Download the code
2. Open in Arduino IDE
3. Select the correct board
4. Upload the code
